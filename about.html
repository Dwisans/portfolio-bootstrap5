About Me

Background

Saya Dwiky Wisan Saputra, saya seorang mahasiswa yang sangat antusias dalam dunia pengembangan web, khususnya Frontend Development. Saya memiliki minat yang besar dalam menciptakan antarmuka pengguna yang menarik, responsif, dan mudah digunakan. Saya selalu bersemangat untuk mempelajari teknologi terbaru dan menerapkannya dalam proyek-proyek pribadi saya.

Skill

Saya menguasai HTML, CSS, JavaScript, dan React, serta menggunakan tools seperti Visual Studio Code, Git, dan Figma. Beberapa proyek terbaru saya termasuk membangun website portofolio pribadi dan aplikasi web responsif untuk tugas kuliah. Tujuan saya adalah meningkatkan keterampilan dan pengetahuan dalam pengembangan web untuk menjadi seorang Frontend Developer yang handal.

Collaboration

Saya terbuka untuk berkolaborasi dalam proyek-proyek pengembangan web, baik sebagai tim maupun individu. Saya pernah bekerja sama dengan teman-teman kuliah dalam mengembangkan aplikasi web untuk tugas kelompok. Jika Anda tertarik untuk bekerja sama atau memiliki pertanyaan, jangan ragu untuk menghubungi saya melalui email atau media sosial.
